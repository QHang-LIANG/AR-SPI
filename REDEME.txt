# MAU-Net for Single-Pixel Imaging (SPI)

This repository contains the implementation of a MAU-Net–based reconstruction framework for single-pixel imaging (SPI).  
The code is tested under the following software environment.

---

## Python Environment

- **Python**: 3.9.21  
- **CUDA**: 11.3  
- **PyTorch CUDA build**: cu113  

---

## Required Python Packages

The following libraries are directly used in the code:

### Core Scientific Computing
- numpy == 1.23.5  
- scipy == 1.9.3  
- pandas == 2.2.3  

### Deep Learning
- torch == 1.12.0+cu113  
- torchvision == 0.13.0+cu113  
- torchaudio == 0.12.0+cu113  

### Image Processing and Visualization
- matplotlib == 3.5.3  
- pillow == 9.2.0  
- opencv-python == 4.11.0.86  
- scikit-image == 0.19.3  

### Data I/O
- openpyxl == 3.1.5  

---

## Notes

- The SPI measurement matrix is generated using NumPy.
- Measured signals are loaded from Excel files via `pandas` and `openpyxl`.
- GPU acceleration is enabled through PyTorch with CUDA support.
- Visualization is handled using Matplotlib.

---

## Reproducibility

To reproduce the experimental results, it is recommended to use the package versions listed above.  
Minor version differences may lead to numerical discrepancies in optimization-based reconstruction.
